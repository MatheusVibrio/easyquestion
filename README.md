## EasyQuestion

Este projeto é um site WEB para gerenciar questões de um sistema educacional, permitindo o cadastro de usuários, questões, respostas, marcadores e outros recursos.

## Requisitos

- **React** 
- **Npm**

## Stack utilizada

**Front-end:** React, Tailwind

## Para rodar:

```bash
npm i
npm run dev
```
